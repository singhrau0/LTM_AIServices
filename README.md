# LTM AI Services Labs

This repository contains beginner-friendly day-wise AI, machine learning, deep learning, computer vision, and NLP lab resources for student revision.

## Repository Structure

| Folder | Purpose |
| --- | --- |
| `Day_01` to `Day_14` | Day-wise lecture notes, notebooks, datasets, and practice files |
| `Assessments_Week_01_Practice` | Week 1 practice assessment datasets and problem statements |
| `Important_Resources` | Revision indexes, lab aggregation, and retrieval-ready metadata |
| `Video_Links` | Day-wise video reference links |
