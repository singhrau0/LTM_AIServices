# Revision Guide

This beginner-level guide is organized for fast student revision before assessments or project work.

## Machine Learning Foundations

- Explain the difference between training, validation, and test data.
- Identify common preprocessing steps for tabular datasets.
- Compare decision trees, random forests, and SVMs.
- Explain bias, variance, overfitting, and underfitting.
- Know when to use pipelines and model persistence.

## Feature Engineering and Model Improvement

- Handle missing values, categorical variables, and scaling.
- Use hyperparameter tuning to improve model performance.
- Read model metrics carefully instead of depending only on accuracy.
- Compare multiple models using a consistent evaluation process.

## Neural Networks

- Understand neurons, layers, activation functions, loss functions, and optimizers.
- Explain the Keras workflow: load data, preprocess, build model, compile, fit, evaluate, predict.
- Know why validation loss can increase while training loss decreases.

## CNN and Computer Vision

- Understand convolution, pooling, flattening, dense layers, and softmax.
- Explain why CNNs work better than dense networks for images.
- Practice image preprocessing and batch preparation.
- Review transfer learning and object detection concepts before later CV work.

## RNN, LSTM, and Sequence Models

- Explain why sequence order matters.
- Understand the vanishing gradient problem in basic RNNs.
- Explain how LSTM gates help retain useful memory.
- Know why padding and truncation are needed for text batches.
- Run `../Day_10/day_10_rnn_lstm_imdb_movie_reviews_lab.ipynb` for a beginner IMDB movie review workflow.

## NLP and Transformers

- Review text cleaning with regex and tokenization.
- Understand embeddings as numerical representations of words or tokens.
- Connect RNN/LSTM limitations to attention mechanisms.
- Review transformer architecture, self-attention, and BERT-style pretraining.
- Run `../Day_12/day_12_beginner_nlp_text_processing_lab.ipynb` for classical NLP preprocessing.
- Run `../Day_13/day_13_beginner_attention_transformer_bert_lab.ipynb` for beginner attention and transformer concepts.
- Run `../Day_14/day_14_beginner_llm_applications_lab.ipynb` for LLM application patterns.
